# 西班牙申根签证办理指南（北京领区）
## 2.1 签证
### 2.1.1 签证所需材料
- **签证申请表**
  - 英文/西班牙文大写印刷体填写（手写/打印均可），本人签字
  - 未成年人需法定监护人签字
  - 表格链接：[Visa-Application-Form-English-Chinese-and-Spanish.pdf](https://web.blscn.cn/beijing/assets/pdf/chinese/Visa-Application-Form-English-Chinese-and-Spanish.pdf)
- **偿付能力证明**
  - 近3-6个月银行对账单（无需存款证明）
- **旅行相关证明**
  - 清晰的旅行计划（交通预定、行程单等）
  - 确认的往返机票预订单（多次入境仅需首次，签证获批后再正式购票）
  - 涵盖全程的住宿证明
- **旅行医疗保险**
  - 申根国境内适用，覆盖紧急医疗、住院、医疗遣返/身故
  - 最低保额3万欧元（或等值人民币）
- **护照及复印件**
  - 护照有效期为预计离开申根国后至少3个月，至少2张空白页
  - 需提供护照所有页复印件，其中护照首页复印件2张
- **身份证明**
  - 中国公民：户口簿原件 + 所有页复印件
  - 非中国公民：有效的中国居留许可
- **证件照**
  - 2张近期彩色照，白色背景
  - 尺寸：26-35mm（宽）× 32-45mm（长）
- **在职相关材料**
  - 任职公司盖章的营业执照复印件
  - 雇主出具的英文证明信（中文需附英文翻译）
    - 要求：公司正式信头纸、加盖公章、签字并标注日期
    - 内容：公司地址/电话/传真、签字人姓名及职务、申请人姓名/职务/收入/工作年限、准假证明
