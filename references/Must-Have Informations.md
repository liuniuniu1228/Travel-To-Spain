# 西班牙旅行必备参考资料 (Reference)

## 1. 官方旅游门户 (Official Tourism Portals)

### 1.1 巴塞罗那 (Barcelona)
* **官方网址**: [Barcelona.com](https://www.barcelona.com/)
* **核心内容**:
    * **热门景点与地标**: 涵盖圣家族大教堂、公园及各类高迪建筑的深度指南。
    * **文化艺术**: 详尽的博物馆清单（Picasso, MNAC等）及参观攻略。
    * **城市体验**: 探索巴塞罗那的不同区域、特色街道（如 La Rambla）及当地传统市场。
    * **实用工具**: 城市公共运输、住宿推荐、实时天气及互联网接入建议。
    * **体育文化**: 专设巴塞罗那足球俱乐部（FC Barcelona）板块，供球迷查询比赛与球场资讯。
    * **预算建议**: 提供当地消费水平参考与省钱小贴士。

### 1.2 马德里 (Madrid)
* **官方网址**: [Visit Madrid (中文版)](https://www.visitmadrid.es/zh)
* **核心特色**: 提供完整的中文界面支持，适合华语游客深度查询。
* **主要板块**:
    * **文化、美食与购物**: 马德里三大博物馆（普拉多等）、地道 Tapas 餐厅及奢侈品/精品店推荐。
    * **家庭与户外**: 为携带儿童的家庭制定的专属计划、体育运动及公园自然空间指南。
    * **深度体验**: 独特的本地体验项目及各种主题游览路线。
* **官方行程建议**:
    * **2日游**: 核心地标速览。
    * **5日游**: 城市深度文化游。
    * **7日游**: 涵盖马德里周边及全城角落的沉浸式体验。

### 3.2 护照丢失应急处理 SOP
1. **立即报警**: 寻找最近的警察局 (Comisaría) 报案，并索取 **报警单 (Denuncia)**。这是补办证件和保险理赔的唯一凭证。
2. **联系使领馆**: 登录“中国领事”App，选择“护照/旅行证”办理业务。紧急情况拨打：+34 699089086 (马德里) 或 +34 932121990 (巴塞罗那)。
3. **准备材料**: 
   * 报警单原件及复印件。
   * 证件照（白底，3.5x4.5cm）。
   * 原护照/身份证复印件（建议手机常备照片）。
4. **办理旅行证**: 旅游者通常办理旅行证（2-4个工作日），仅用于回国。
5. **后续操作**: 拿到旅行证后，联系航空公司或 Renfe 更新证件信息。
