
## 📌 项目概述
本仓库包含一份结构化的 `skill.md` 文档，旨在为 AI Agent 提供关于**西班牙申根签证办理**及**西班牙当地生存指南**的深度知识。

## 🎯 核心功能
* **签证导航**：提供北京领区 BLS 申请的精准材料清单和预约入口。
* **工具推荐**：针对西班牙本土市场（如 Glovo, Bolt）的 App 选型建议。
* **危机处理**：内置护照丢失、医疗急救等紧急情况的标准化处理流程（SOP）。
* **领事保护**：集成官方联系方式与使馆地理位置信息。

## 🚀 集成指南

### 1. 手动导入至 GPTs / 知识库
1.  下载本仓库的 `skill.md`。
2.  在 Agent 的 **Knowledge（知识库）** 中上传该文件。
3.  在 **System Prompt** 中添加：
    > "请参考 `skill.md` 中的内容为用户提供西班牙服务。当涉及‘签证’、‘护照丢失’或‘当地出行’时，严格遵循文档中的 SOP。"

### 2. 开发者模式（RAG 优化）
* **分段依据**：按二级标题（`##`）切分。
* **检索权重关键词**：西班牙签证、护照补办、BLS预约、紧急电话、马德里/巴塞罗那。
